# Nori

Nori is a native Windows virtual-machine app for macOS. It uses Apple’s Virtualization framework and can obtain Windows installation media directly from Microsoft’s official download service.

## Download

Download the newest build from [GitHub Releases](https://github.com/LeBronFanCW/NoriVM/releases/latest).

Nori 0.4 and newer check this repository’s signed Sparkle feed automatically. Every update is verified with an Ed25519 signature before installation.

Windows installation media and licenses are provided by Microsoft and are not included with Nori. Apple silicon supports Windows 11 ARM guests; Intel builds support compatible x64 guests.
