## Nori 0.4

- Adds secure automatic updates through GitHub Releases and Sparkle.
- Verifies every update with an embedded Ed25519 public key before installation.
- Adds **Check for Updates…** to the Nori menu and Settings.
- Checks for updates automatically once per day and can install downloaded updates automatically.
- Includes the NVMe storage, stable networking, safer shutdown, and performance improvements from Nori 0.3.
